# Production-Grade Breast Cancer Detection Framework

An end-to-end Machine Learning implementation utilizing the Wisconsin Breast Cancer Dataset to identify and classify malignant cellular samples. This project is built using isolated pipeline patterns to eliminate data leakage and features an interactive web dashboard.
